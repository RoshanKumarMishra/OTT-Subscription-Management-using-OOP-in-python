# OTT-Subscription-Management-using-OOP-in-python

# For File Persistence & DB:
Engineered a persistent data layer using Python’s sqlite3 to securely store user profiles and track active subscriptions, eliminating data loss across application resets.

# For Design & Architecture (Exceptions):
Architected robust error-handling pipelines by implementing custom OOP exception classes, reducing system crashes and streamlining edge-case management for failed payments.

# For Unit Testing:
Developed comprehensive test suites using the pytest framework, achieving high code coverage and ensuring system reliability across membership tier upgrades.

# For Notification Simulation:
Built an automated notification logging system that monitors user lifecycles and flags expiring accounts, simulating real-world cron-job logic.
